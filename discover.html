Question-1: What are the different ways to select an element in the DOM?

By using getElementById, getElementsByClassName, getElementsByTagName, or querySelector and querySelectorAll one can select an element in the DOM

Question-2: What is the difference between innerHTML, innerText, and textContent?

innerHTML gives you the HTML markup inside an element, innerText returns only the visible text, and textContent retrieves all text within the element

Question-3: What is event delegation in the DOM?

Event delegation means adding one event listener to a parent element instead of multiple listeners to each child.

Question-4: What is event bubbling in the DOM?

Event bubbling is when an event triggered on a child element also fires on its parent elements, moving upward through the DOM tree.
